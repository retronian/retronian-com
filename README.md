# Retronian

👾 **One OS, All Handhelds.**

Retro gaming, handheld game consoles, custom OS, and 3D printed accessories.

## Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`

## Deploy

This site is deployed via GitHub Pages with a custom domain (`retronian.com`).

Push to the `main` branch to deploy.

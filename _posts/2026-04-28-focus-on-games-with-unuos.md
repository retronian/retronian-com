Retro handheld fans, are you overwhelmed by a huge number of retro games?

Do you just want to relax with nostalgic games, but find yourself worn out by managing tons of games and configuring all kinds of emulators?

Let's just focus on the games.

[UnuOS](https://github.com/retronian/UnuOS) now supports **Game Focus Mode**.

When you boot the device, the only things shown on screen are the games you selected. No settings or anything else unnecessary.

![Game Focus Mode](/assets/images/game-focus-en.png)
*Game Focus Mode*

It's like the **Flavor Concentration Booth** at Ichiran ramen. On a game console, let's focus only on games.

## How to Use It

[UnuOS](https://github.com/retronian/UnuOS) is an OS derived from MinUI, so it has a Collection feature. All you need to do is prepare a Collection named `Selection.txt`.

If a Collection named `Selection.txt` exists, only the games inside it will be displayed, and everything else, such as settings and recently played software, will be hidden.

For details, see the page below.

[Game Focus Mode](https://github.com/retronian/UnuOS/wiki/Game-Focus-Mode)
